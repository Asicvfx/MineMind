# 🎭 ТЗ на мем-фото для MoodMode

> Бриф для поиска картинок. Каждая картинка вылетает на экран на игровое событие.
> Два набора: **Chill** (лёгкое, чтоб улыбнулся) и **Chaos** (на максималках, чтоб орал).
> Файлы кидать в `frontend/public/memes/chill/` и `frontend/public/memes/chaos/`
> с именем по событию: `lose_fast.png`, `win.png` и т.д.

## Общие требования к картинкам

- Формат: **PNG или JPG**, желательно квадрат или близко (1:1 … 4:3)
- Размер: 400–800px по большей стороне, вес < 400 KB
- На картинке **уже может быть надпись** (если мем с текстом) — ок
- Если картинка без текста — мы добавим подпись из списка ниже сами
- Стиль: яркие, читаемые, выразительные лица/эмоции — без мелких деталей
  (картинка вылетает на 2.5 секунды, должна "читаться" мгновенно)

## Как вылетает (анимация)

| | Chill 😎 | Chaos 🔥 |
|--|---------|---------|
| Вход | плавно выезжает снизу, лёгкий наклон | **с прокруткой** — крутится 1.5 оборота, влетает с прыжком |
| Размер | средний, ~180px | **большой**, ~320px, чуть пульсирует |
| Доп. эффекты | мягкое свечение | тряска экрана + цветная вспышка + громкий звук |
| Время на экране | 2.5 сек | 2.5 сек, агрессивно |
| Звук | мягкий «поп» | громкий: взрыв / фанфары / вой |

# 📋 СПИСОК МЕМОВ ПО СОБЫТИЯМ

---

## 1. `game_start` — игрок сделал первый клик

**Chill** 😎
- **Смысл фото:** человек спокойно разминает пальцы / надевает наушники / делает глоток кофе перед работой. Расслабленная готовность.
- **Надпись:** «Поехали. Без суеты 🙂»

**Chaos** 🔥
- **Смысл фото:** кто-то резко надевает чёрные очки, за спиной взрыв / огонь. Энергия «it's go time». Или тип с лицом «сейчас будет жарко».
- **Надпись:** «ПОЕХАЛИ. Не позорься.»

## 2. `flag_correct` — поставил флаг точно на мину

**Chill** 😎
- **Смысл фото:** человек одобрительно кивает / показывает мягкий палец вверх / лёгкая улыбка «неплохо».

# 🎯 Резюме для поиска

Всего нужно **20 картинок** (10 событий × 2 режима).

**Минимальный набор** (если времени мало) — самые частые и эффектные события:
- `win` (chill + chaos)
- `lose` (chill + chaos)
- `lose_fast` (chill + chaos)
- `win_fast` (chill + chaos)
- `near_loss` (chill + chaos)
→ **10 картинок** покроют 90% игрового опыта.

**Куда класть:**
```
frontend/public/memes/
  chill/
    win.png  lose.png  lose_fast.png  win_fast.png  near_loss.png  ...
  chaos/
    win.png  lose.png  lose_fast.png  win_fast.png  near_loss.png  ...
```

Система сама подхватит файл. Если файла нет — покажет эмодзи (уже работает).
После добавления файлов — просто refresh, без ребилда.
