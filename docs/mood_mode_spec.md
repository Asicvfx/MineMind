# 🎭 MoodMode — Emotion System для MineMind

## Концепция

**MineMind = brain-training Minesweeper, но с настроением.**
Игрок сам выбирает, *как* игра с ним разговаривает. Одна и та же механика Сапёра —
три совершенно разных эмоциональных опыта.

Это не "клоунада поверх игры" — это **продуктовая фича про удержание (retention)**:
人 возвращаются не ради клеток, а ради *вайба*. Жюри увидит, что мы думаем про
эмоцию пользователя, а не только про логику.

---

## Три режима

### 🧘 Focus Mode — «для вундеркиндов»
Для тех, кто хочет тишину и концентрацию. Решает задачу как шахматист.

- Фон: тихий lo-fi / ambient / джаз (или генеративный эмбиент через Web Audio)
- Звуки: минимальные — мягкий «тук» на клик, ничего лишнего
- Реакции: **молчит**. Максимум сухие «Верно» / «Точный ход» от AI Coach
- Анимации: спокойные, без тряски
- UI: чище, меньше декора, больше воздуха
- Вайб: *библиотека, наушники, поток*

### 😎 Chill Mode — «для кайфа»
Для тех, кто хочет лёгкое удовольствие от игры. Дефолтный режим.

- Фон: мягкий приятный бит (опционально)
- Звуки: приятные «поп» при открытии, мягкий «дзынь» на флаг
- Реакции: дружелюбные, тёплые фразы (toast снизу, маленькие)
- Анимации: плавные пружинки, мягкое свечение
- Вайб: *выходной, кофе, расслабон*

**Примеры фраз (Chill):**
- Поставил правильный флаг → «Чисто. Хороший read 👍»
- Победа → «Изящно. Ты решал, а не угадывал.»
- Победа быстро → «Скорость + точность. Уважение.»
- Проигрыш → «Минус вайб, плюс опыт. Го ещё раз.»
- Чуть не проиграл → «Пронесло 😮‍💨»

### 🔥 Chaos Mode — «угар»
Для тех, кто хочет, чтобы игра орала на них и стебала. Максимальный движ.

- Фон: хайповый бит / драм-н-бэйс
- Звуки: громкие — взрыв с эхом, фанфары, «вой» при быстром проигрыше
- Реакции: **жёсткие мем-фразы**, всплывают КРУПНО посреди экрана
- Анимации: экран трясётся при ошибке, вспышки, цветной глитч
- Toast: большие, с эмодзи, агрессивно влетают
- Вайб: *стрим, друзья орут, мемы*

**Примеры фраз (Chaos):**
- Поставил правильный флаг → «ОТКУДА ЗНАЛ?? Ты подсматриваешь 🤨»
- Победа → «Ну окей, читер. Засчитано 🏆»
- Победа быстро → «Хмм... подозрительно быстро. Ну ладно, красава.»
- Проигрыш за <10 сек → «Брат. Даже двухлетка так не сливает 💀»
- Проигрыш → «Скилл ишью. Это не игра виновата.»
- Чуть не проиграл → «ОДИН клик от смерти. Сердечко ещё бьётся?»
- Долгая серия без ошибок → «Big brain detected 🧠⚡»

> ⚠️ Жёсткость фраз настраивается (см. ниже). Версия с матом — отдельный «18+» тон.

## Триггеры реакций (game events)

| Событие | Когда срабатывает |
|---------|-------------------|
| `game_start` | первый клик |
| `flag_placed` | поставлен флаг |
| `flag_correct` | в конце игры — флаг был на мине |
| `flag_wrong` | в конце игры — флаг был НЕ на мине |
| `reveal_safe` | открыта безопасная клетка |
| `near_loss` | открыта клетка с числом ≥ 5 (было близко) |
| `streak` | открыто N клеток подряд без флагов/ошибок |
| `win` | победа |
| `win_fast` | победа быстрее порога (easy<30s, medium<90s, hard<180s) |
| `lose` | проигрыш |
| `lose_fast` | проигрыш быстрее 10 секунд |
| `ai_hint_used` | использована подсказка AI Coach |

## Что технически меняется между режимами

| Аспект | Focus 🧘 | Chill 😎 | Chaos 🔥 |
|--------|---------|---------|---------|
| Фоновая музыка | lo-fi тихо | мягкий бит | хайп-бит |
| Звуки действий | минимальные | приятные | громкие + эхо |
| Текст-реакции | нет | дружелюбные toast | жёсткие мемы крупно |
| Анимации | спокойные | пружинки | тряска + глитч |
| Попап на win/lose | тихий modal | обычный modal | большое окно с ржачкой |
| Интенсивность визуала | −20% | базовая | +30% (вспышки) |

## Архитектура (как строим)

```
composables/
  useMood.ts        — текущий режим (persist в localStorage), геттеры
  useSound.ts       — Web Audio API: синтез звуков (БЕЗ файлов = без копирайта)
  useReactions.ts   — словарь фраз по {mood × event}, выбор случайной фразы
  useGameAudio.ts   — фоновая музыка (play/pause/volume по режиму)

components/
  mood/
    MoodPicker.vue      — селектор 3 режимов (на /play сверху + в профиле)
    MoodReaction.vue    — toast/попап с фразой, стиль зависит от режима
    ChaosOverlay.vue    — экранные эффекты для Chaos (тряска, вспышки)

stores/
  settings.ts       — mood, soundEnabled, musicEnabled, phraseTone
```

### Звук — без копирайта
- **Звуковые эффекты** (клик, взрыв, победа) — генерируем через **Web Audio API**
  (осцилляторы + envelope). Ноль файлов, ноль копирайта, ~150 строк кода.
- **Фоновая музыка** — два пути:
  - **(A)** Royalty-free треки (Pixabay Music / Kevin MacLeod CC-BY / FreePD) —
    ты скачиваешь 2-3 mp3, я подключаю. Звучит «по-настоящему».
  - **(B)** Генеративный ambient через Web Audio (медленные пады). Бесплатно,
    но это не «джаз», а скорее «космический эмбиент».

### Фразы — структура
```ts
const REACTIONS = {
  chaos: {
    win_fast: [
      'Хмм... подозрительно быстро. Ну ладно, красава.',
      'Ты не играл, ты ЧИТИЛ. Засчитано 🏆',
    ],
    lose_fast: [
      'Брат. Даже двухлетка так не сливает 💀',
      'Speedrun to explosion 💥 Поздравляю?',
    ],
    // ...
  },
  chill: { /* ... */ },
  focus: { /* почти пусто */ },
}
```
Каждый раз берётся **случайная** фраза из массива → не приедается.

## Объём работ (оценка)

| Часть | Время |
|-------|-------|
| `useMood` + `settings` store + `MoodPicker` UI | 30 мин |
| `useSound` (Web Audio синтез эффектов) | 40 мин |
| `useReactions` + словарь фраз (3 режима × ~10 событий) | 30 мин |

| `ChaosOverlay` (тряска, вспышки, глитч) | 30 мин |

| Фоновая музыка (если royalty-free файлы) | 20 мин |
| **Итого** | **~3.5 часа** |

## Решения, которые нужны от тебя

1. **Тон фраз**
   - `Clean` — без мата, дружелюбно-стёбно (безопасно для жюри)
   - `Edgy` — резко, но без мата («скилл ишью», «иди поучись») ← рекомендую
   - `18+` — с матом (рискованно для официальной сдачи)
   - можно сделать **переключатель тона** в настройках (default = Edgy)

2. **Фоновая музыка**
   - `(A)` ты находишь 2-3 royalty-free трека → я подключаю
   - `(B)` генеративный ambient через Web Audio (без файлов)
   - `(C)` пока без музыки, только звуковые эффекты — добавим музыку позже

3. **Где показывать MoodPicker**
   - на странице `/play` сверху (быстрый доступ)
   - + опционально в профиле как «дефолтный режим»
